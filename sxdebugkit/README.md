# sx-debug-kit

有助于 Android 开发的一些测试、调试工具。

主要内容如下：

- FloatLog
 - 悬浮窗的形式显示日志
- HttpInterceptor
 - 拦截网络请求结果，直接在设备查看
- ActivityNameTracker
 - 显示前台应用包名及 Activity 名
- MemoryTracker
 - 显示内存使用情况
 - 参考 LeakCanary
- MethodRunTime
 - 方法耗时 参考：https://github.com/JakeWharton/hugo
 - 事件耗时 参考：https://github.com/NikitaKozlov/Pury
